# Restaurant-Order-Management-System-
Project môn CNPM, xây dựng hệ thống quản lý đặt món tại bàn

Luồng nghiệp vụ tổng quát:
Khách ngồi vào bàn
Quét QR trên bàn
Hệ thống mở website/app hiển thị menu đúng cho bàn đó
Khách chọn món → tạo order
Order chuyển vào: màn hình bếp/barista
Khi món hoàn thành: nhân viên mang ra bàn
Khách thanh toán: online hoặc tại quầy
Hóa đơn đóng lại

Dự án chia thành 4 màn hình chính:
Customer UI (order)
Kitchen UI (xem order/món cần làm)
Cashier UI (thanh toán)
Admin/Manager UI (menu – báo cáo)
